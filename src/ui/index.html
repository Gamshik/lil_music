SoundCloud Desktop

Desktop shell уже живой, а поиск и playback идут через native bridge.

Поиск работает через `api-v2.soundcloud.com`, а карточки треков приходят в web-layer уже из C++ сценариев. Это базовая площадка для дальнейшего desktop-клиента.
